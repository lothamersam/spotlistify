# spotlistify
Website adding additional functionality to Spotify playlists.

- Spotify playists from upcomping and old concerts

Using python Flask web framework.

## To depoloy locally:
1. Clone repo.
2. Create a python virtual environment in project root directory.
3. Run `pip install -r requirements.txt` to install dependencies

__NOTE__: If any dependcies are installed, run `pip freeze > requirements`
